This repository includes some rough mockup code for profile-editing.

Instructions:
 * Install CiviCRM (Note: only tested with trunk but may work with 4.2)
 * Clone this git repository; put the resulting directory somewhere in your CiviCRM extension directory
 * Execute "packages.sh" to download dependencies
 * Enable the extension at /civicrm/admin/extensions?reset=1 (click refresh to see it)
 * Try it out at /civicrm/profile-editor
